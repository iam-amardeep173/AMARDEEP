Plant Disease Recognition System

The Plant Disease Recognition System is an AI-powered web application designed to help farmers and agricultural professionals detect plant diseases from leaf images, preventing crop losses, ensuring early treatment, and improving agricultural productivity.

Project Overview

Objective: The Plant Disease Recognition System uses deep learning models to detect plant diseases based on leaf images, allowing farmers to identify and treat diseases early, thereby improving crop health and yield.

Dataset & Preprocessing

The dataset consists of over 87,000 images classified into 38 categories, including both healthy and diseased plant leaves. The dataset is divided into training, validation, and test sets. Images are resized to 128×128 pixels for uniform input processing.

Model Development & Training

A Convolutional Neural Network (CNN) is used for classification. Data augmentation was applied to improve model generalization, and Adam optimizer was used for training the model. The model achieved high accuracy in detecting plant diseases.

Model Performance

The CNN model achieved a training accuracy of 94.59% and a validation accuracy of 94.58%. Key results include:

  • Tomato Yellow Leaf Curl Virus - 99% Accuracy
  • Grape Healthy - 99% Accuracy
  • Corn Common Rust - 99% Accuracy

Web Application (Streamlit)

The model is deployed using Streamlit, providing an interactive web application for non-technical users, allowing them to upload leaf images for disease predictions.

Features

  • Home Page - Introduction to the system.
  • About Page - Details of the dataset and model training process.
  • Disease Recognition Page - Upload leaf images for disease prediction with confidence scores and disease name.
  • Interactive Snowfall Animation - Enhances the user experience after predictions.

How It Works

1. User uploads a plant leaf image.
2. Image is processed and resized to 128×128 pixels.
3. The CNN model predicts the disease and outputs the disease name and confidence score.

Tools & Technologies Used

  • Programming Language: Python
  • Frameworks: TensorFlow/Keras
  • Web Framework: Streamlit
  • Development Environment: VSCode, Google Colab
  • Image Preprocessing: OpenCV
