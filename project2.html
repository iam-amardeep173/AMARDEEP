Object Detection

This project focuses on Object Detection, a computer vision technique that identifies and localizes objects in images or videos. It involves a combination of image classification (identifying object categories) and object localization (bounding box detection).

Project Overview

Objective: Object detection involves recognizing and localizing objects of interest within an image. The goal is to assign correct class labels and precise bounding box coordinates to detected objects.

Algorithm Used - YOLO (You Only Look Once)

The project implements the YOLOv3 algorithm, which is a real-time object detection model that uses a single-shot detection approach. Key features of YOLO include:

  • Darknet as Backbone: The neural network framework for YOLO.
  • Single-stage detection: Unlike RCNN-based models, YOLO doesn’t use a region proposal network.
  • Use of anchor boxes: Allows detecting multiple objects in the same image.
  • Non-Maximum Suppression (NMS): Eliminates overlapping bounding boxes to retain the most accurate one.

Applications of Object Detection

  • Face Detection
  • Text and Logo Detection
  • Biometric Systems
  • Medical Imaging
  • Self-Driving Cars
  • Video Surveillance

Methodology

  • Intersection Over Union (IoU): Measures accuracy by comparing predicted and actual bounding boxes.
  • Non-Max Suppression (NMS): Removes redundant bounding boxes with low confidence scores.
  • Prediction Equations: Uses sigmoid activation to refine bounding box coordinates.

Tools & Technologies Used

  • Programming Language: Python 3.7
  • Frameworks & Libraries: PyTorch, OpenCV
  • Development Environment: Google Colab, VSCode
  • Annotation Tool: labelImg

Results

Achieved a mean Average Precision (mAP) of 0.59 for 80 classes. The model performed well on high-resolution images but struggled with low-resolution images.

Future Work
