Sign Language Recognition

Developed a system to recognize Indian Sign Language gestures from real-time video and convert them into text using deep learning models.

Project Overview

Objective: To develop a computer application capable of recognizing hand gestures in real-time for Indian Sign Language (ISL) and converting them into text.

Dataset: The INCLUDE-50 dataset was used, containing 50-word signs, 0.27 million frames, and 4,287 videos. The dataset was enhanced with additional videos created for the project.

Key Components:

  • Preprocessing: Video frames were captured and flipped to analyze both original and mirrored versions.
  • Keypoint Extraction: MediaPipe was used to extract 21 keypoints per hand and 25 keypoints from the upper body (shoulders, wrists, facial landmarks).
  • Hand Assignment: Used wrist keypoints to ensure accurate identification of left and right hands.
  • Parallel Processing: Processed multiple videos simultaneously to speed up preprocessing.

Data Augmentation: Applied rotation (+7° and -7°) and flipping transformations to improve model generalization.

Models Used:

  • LSTM: 8 layers, bidirectional, hidden size 256, dropout of 0.2. Achieved 27% accuracy.
  • Transformer: 4 hidden layers, hidden size 512, 8 attention heads, positional embeddings. Achieved 94.64% accuracy, outperforming LSTM by a significant margin.

Optimization: Used the Adam optimizer and implemented early stopping to improve performance.

Results: The Transformer model achieved 99% training accuracy and 94.64% test accuracy.
